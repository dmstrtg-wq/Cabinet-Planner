Commitment to Access

Accessibility Statement

Last updated: June 2025

My Cabinet Planner is committed to ensuring digital accessibility for people with disabilities. We are continually working to improve the user experience for everyone and to apply relevant accessibility standards.

Our Conformance Status

We are actively working toward conformance with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. The following measures have been implemented:

  • Visible keyboard focus indicators on all interactive elements
  • Skip navigation link to bypass repeated content
  • ARIA labels on all icon-only buttons
  • ARIA dialog roles and attributes on all modal windows
  • Properly associated form labels on all input fields
  • Sufficient color contrast on primary text and interface elements
  • Semantic HTML landmarks (main, aside, footer, nav)

Known Limitations

My Cabinet Planner is a complex graphical design tool. The following areas have known accessibility limitations that we are working to address:

  • Floor Plan canvas: The interactive drag-and-drop floor plan is rendered on an HTML canvas element. Canvas content is not natively accessible to screen readers. Cabinet data is available in text form via the cabinet list panel on the right side of the screen.
  • Elevation view: The wall elevation canvas is a graphical representation and is not screen-reader accessible. Cabinet placement information is available in text form via the cabinet list.
  • 3D View: The 3D model is rendered using WebGL and is not screen-reader accessible. It is provided as an optional visual aid only.

We recognize that these limitations may present barriers. If you need assistance accessing any feature of My Cabinet Planner, please contact us and we will do our best to accommodate your needs.
